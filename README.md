## 🚀 Features

- **Create & Edit Notes:** Easily jot down your thoughts, ideas, and tasks.
- **Share:** Share your notes with others seamlessly.
- **Organize:** Keep your notes structured and accessible.
- **Responsive Design:** Take your notes on the go with a mobile-friendly UI.

## 🛠️ Tech Stack

This project is divided into two main environments:

- **Client:** React, React Router DOM, standard CSS.
- **Server:** Node.js, Express.js, MongoDB (Mongoose), CORS.

## 📁 Project Structure

```text
shareyournotes/
├── Client/       # React Frontend (UI, Routing, State)
└── Server/       # Express Backend (API, DB connection, Routes)
```

## ⚙️ Getting Started

### Prerequisites

Ensure you have the following installed on your local machine:

- [Node.js](https://nodejs.org/) (v14 or higher recommended)
- npm or Yarn

### Installation & Setup

1. **Setup the Server:**

   ```bash
   cd Server
   npm install
   npm start
   ```

2. **Setup the Client:**
   Open a new terminal window/tab:
   ```bash
   cd Client
   npm install
   npm start
   ```
